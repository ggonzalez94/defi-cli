## Feature snapshot

- Lending and rates from Aave + Morpho (with DefiLlama fallback paths).
- Yield ranking across provider-backed routes.
- Bridge quotes via Across and LiFi.
- Swap quotes via 1inch and Uniswap.
- Chain/protocol discovery and canonical ID resolution.
